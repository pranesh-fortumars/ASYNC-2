## 🚀 SYSTEM INITIATION

This repository contains the source code for my premium, interactive 3D personal portfolio. Built with a dark, cyberpunk-inspired aesthetic, the site leverages cutting-edge web technologies to deliver an Apple-level user experience.

### 🌟 Key Features
- **3D Particle Universe**: Interactive `react-three-fiber` hero background.
- **Skills Galaxy**: A 3D orbital system representing technical expertise.
- **Tech Sphere**: Rotating 3D cloud of technologies.
- **Cinematic Experience**: Smooth scrolling via `Lenis` and fluid transitions via `Framer Motion`.
- **Interactive UI**: Magnetic buttons, 3D tilt cards, and glassmorphism layouts.
- **Futuristic Terminal**: A contact form simulating an encrypted payload delivery.

## 🛠️ INSTALLATION & LAUNCH

To run this application locally, ensure you have Node.js installed, then execute the following commands:

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/portfolio.git

# 2. Navigate into the directory
cd portfolio

# 3. Install dependencies
npm install

# 4. Initiate the development server
npm run dev
```

> **Target Acquired:** Open `http://localhost:3000` in your browser to view the application.

## 🗄️ ARCHITECTURE & TECH STACK

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Framework** | `Next.js 15 (App Router)` | Core React framework for SSR and routing. |
| **Styling** | `Tailwind CSS v4` | Utility-first styling with a custom dark theme. |
| **Animations** | `Framer Motion` | Complex UI transitions, page loading, and gestures. |
| **3D Graphics** | `Three.js` & `R3F` | Canvas rendering for the Particle Universe and Tech Sphere. |
| **Interactivity** | `Lenis` | Custom smooth scroll physics. |
